forceMwApiLoad()

Forces mediawiki.api module to load During Selenium test suite runs, some random tests would consistently break because mediawiki.api would get stuck at "loading" status. This forces mw.Api to load by calling it directly, bypassing any cache or mw.loader controll. With any luck, this should not happen in production.
